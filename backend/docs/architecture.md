# Backend Architecture

## System Flow

React Frontend
    ↓
FastAPI Backend
    ↓
PostgreSQL Database
    ↓
AGMARKNET Dataset

---

## Responsibilities

Frontend:
- Display prices
- Charts
- Maps
- Forecasts

Backend:
- Fetch data
- Process data
- Serve APIs
- Connect database

Database:
- Store historical prices
- Store mandi information
- Store users
- Store alerts

Data Source:
- AGMARKNET datasets
